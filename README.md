🧺 LaundryWallah – Premium Laundry Service Website

LaundryWallah is a modern and responsive landing page for a laundry service business.
It showcases services, branding, and a call-to-action section for booking laundry services.



🚀 Features

✅ Responsive Navigation Bar

✅ Clean Hero Section Layout

✅ Modern UI with Soft Blue Theme

✅ Call-to-Action Button with Hover Effects

✅ Flexbox Layout for Proper Alignment

✅ Optimized Hero Image Sizing
